# Podcast Strategy — Own Your Mind

AI-generated podcast using Rob's cloned voice (ElevenLabs) as host, with a second AI voice as expert analyst. Repurposes existing project reviews and articles into audio format for podcast distribution and on-site embedding.

Updated: 2026-03-09

---

## Why Podcast

- **Leverage existing content.** 34 project reviews already written. Podcast repurposes that research into a second distribution channel.
- **Compound content.** Each episode lives on the project page (time-on-page SEO signal), in podcast directories (passive discovery), and as newsletter content. One piece of work, three channels.
- **Brand moat.** Rob's cloned voice carries the practitioner authenticity. Harder to replicate than written content.
- **Fits 30% time constraint.** Agent Zero scripts, ElevenLabs generates, Rob reviews and publishes. ~20 min per episode.

---

## Format

### Episode Types

| Type | Duration | Content Source | Priority |
|------|----------|---------------|----------|
| **Project Deep-Dive** | 15-20 min | Individual project review | Primary — start here |
| **Comparison Episode** | 20-25 min | Tokenomics comparison articles | Secondary — once format proven |
| **Theme Episode** | 15-20 min | Why DeAI thesis articles | Tertiary — for broader audience |

### Voices

| Role | Voice | Source |
|------|-------|--------|
| **Host (Rob)** | Cloned voice | ElevenLabs — voice clone created |
| **Co-host (Rachel)** | AI voice | ElevenLabs — Rachel McGrath |

### Episode Structure (Project Deep-Dive)

```
0:00-1:00   Intro — What we're covering, why it matters
1:00-4:00   What is [Project]? — Core technology, what it does
4:00-9:00   Freedom Score — Decentralisation assessment, key dimensions
9:00-14:00  Returns Score — Tokenomics, value accrual, liquidity
14:00-17:00 Honest Assessment — What works, what doesn't, key risks
17:00-18:00 Outro — Key takeaways, where to find the full review
```

## Production Workflow

```
Agent Zero: Write script from existing review
     |
Rob: Review script, adjust for voice/tone
     |
ElevenLabs: Generate audio (Rob voice + Rachel voice)
     |
Edit: Combine clips, add intro/outro music bed
     |
Upload: Podcast host → auto-distributes to all platforms
     |
Embed: Add player to project page on site
```

**Estimated time per episode:** 15-20 min (review + upload + embed)

## Site Integration

### Audio Player Embed

Embed podcast player on project pages for visitors who prefer audio. Options:
1. Spotify embed iframe
2. Native HTML5 audio player with self-hosted MP3
3. Podcast host widget (Transistor/Anchor)

Decision: TBD after pilot — depends on hosting choice.

### Implementation Notes

- Add optional `podcastUrl` or `podcastEmbed` field to project frontmatter
- ProjectLayout conditionally renders audio player when field present
- Player sits below the hero/scores section, above the editorial content
- Fallback: no player if no podcast episode exists for that project

## Pilot Plan

### Phase 1: Validate (2-3 episodes)

1. **Produce 2-3 pilot episodes** — start with project deep-dives
2. **Embed on project pages** — test whether visitors actually play them
3. **Do NOT launch to podcast directories yet** — validate on-site engagement first

- Monthly episode cadence (matches newsletter)
- Script generated by Agent Zero from existing/updated reviews
- Quarterly theme episodes from Why DeAI articles
