## How to use this directory

This is a starting point, not investment advice. Projects are included based on whether they have a working product or credible testnet. Market cap and hype are not criteria for inclusion.

Each project with a full review link has been assessed using the [six-point framework](/projects/) with a freedom score. Projects without review links are on the roadmap.
