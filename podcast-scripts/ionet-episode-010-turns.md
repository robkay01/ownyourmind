## TURN 03 — BOBSKI
```
So what does io.net actually do?
```

## TURN 04 — RACHEL
```
It's a GPU marketplace. You supply GPUs, you rent GPUs, and io.net sits in the middle taking a cut. The GPUs come from independent suppliers across 138 countries -- that's the "decentralised" part. Everything else -- orchestration, matching, billing, verification, and the node software -- is run by io.net. It's built on Solana, raised 30 million in a Series A led by Hack VC at a billion-dollar valuation, and launched on Binance Launchpool in June 2024.
```

## TURN 05 — BOBSKI
```
And the founding story is complicated.
```

## TURN 06 — RACHEL
```
Ahmad Shadid founded io.net in 2022, then resigned as CEO two days before the token launch amid allegations about his background and questions about inflated GPU metrics. Tory Green took over, then moved to chair of the foundation. Gaurav Sharma from Binance became CEO. Three CEOs in under two years. The founder's subsequent project also faced misrepresentation allegations.
```

## TURN 07 — BOBSKI
```
What about the revenue? Because that's the headline number.
```

## TURN 08 — RACHEL
```
Revenue has been growing rapidly quarter-on-quarter and is significant for a DePIN project by any standard. Millions of compute hours per month. The confidential compute offering with Intel TDX and NVIDIA hardware-based attestation is a genuine differentiator too. Check the site for current revenue figures.
```

## TURN 09 — BOBSKI
```
But the GPU numbers are the problem.
```

## TURN 10 — RACHEL
```
Exactly. io.net markets hundreds of thousands of registered GPUs. The daily average of verified, active GPUs is a tiny fraction of that. Single-digit percentage utilisation. And it gets worse -- in April 2024, a massive Sybil attack saw fake GPUs attempting to connect to farm airdrop rewards. An RTX 4090 was reportedly split into infinite virtual GPUs. The project's own CSO admitted, quote, "we messed up."
```

## TURN 11 — BOBSKI
```
Freedom Score is an F grade. What's driving that?
```

## TURN 12 — RACHEL
```
Governance is the lowest dimension. No DAO, no on-chain voting, no governance forum, no proposals mechanism. Token holders have zero governance rights. All decisions made by the team and foundation. Open source transparency also scores poorly because the core platform code is closed source. They brand themselves as "The Open Source AI Infrastructure Platform" -- but the IO Worker binary, the orchestration engine, the matching engine, the billing system, all proprietary. Only a handful of public repos.
```

## TURN 13 — BOBSKI
```
And for context, Akash publishes everything.
```

## TURN 14 — RACHEL
```
Exactly. On the positive side, data sovereignty scores reasonably well thanks to the confidential compute VMs. Intel TDX plus NVIDIA H100s and H200s provide hardware-based data protection during processing. Users can verify attestation reports. For sensitive AI workloads, that genuinely matters.
```

## TURN 15 — BOBSKI
```
Returns Score is a D grade. The revenue is real -- what's dragging the score down?
```

## TURN 16 — RACHEL
```
Supply dynamics scores poorly. Seed investors hold 12.5 percent, team 11.3 percent, Series A 10.15 percent -- a third of supply in insider hands. Unlocks run through 2028. The burn mechanism exists -- 50 percent of platform revenue gets burned -- but the annual buy pressure is dwarfed by hundreds of millions in unlocking insider tokens. Value accrual also scores poorly because the mechanism linking revenue to token value is poorly defined.
```

```
The token is deep below its all-time high.

## TURN 18 — RACHEL
```
The token has crashed from its listing price. Investors who bought at the billion-dollar Series A valuation are deeply underwater. Revenue sustainability is the genuine bright spot -- real customers paying for real GPU time. But the competitive position is squeezed. Akash is more decentralised, more transparent, and cheaper on H100s. Centralised providers are more reliable with SLAs.
```

## TURN 19 — BOBSKI
```
Honest assessment -- what works, what doesn't?
```

## TURN 20 — RACHEL
```
What works: significant annualised revenue with strong growth. Confidential compute is ahead of most DePIN competitors. 85-plus partnerships. Real customers paying for real compute.

What doesn't: closed-source core, no governance, inflated GPU metrics, founder scandal, massive price decline. Strip away the DePIN branding and you have a centralised GPU marketplace with a token. If you want decentralised GPU compute, Akash is the more honest option. If you just want cheap GPUs, Lambda Labs and Vast.ai don't require a token.
