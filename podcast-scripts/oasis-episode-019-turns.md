```
What does Oasis do at the technical level?
```

## TURN 04 — RACHEL
```
Oasis is a Layer 1 built from the ground up for confidential computing. Smart contracts process data that stays encrypted even from the nodes running them. The network has a two-layer design. The Consensus Layer runs CometBFT proof-of-stake with 120 validator slots. The ParaTime Layer runs parallel execution environments. Sapphire is the flagship -- a confidential EVM-compatible runtime using Intel SGX trusted execution environments. Data enters encrypted, is decrypted only inside the hardware enclave, processed, and re-encrypted before output. It is the only production confidential EVM in crypto.
```
