```
What does Ora actually do?
```

## TURN 04 — RACHEL
```
It enables smart contracts to access AI models -- Llama, Stable Diffusion, DeepSeek, Mistral -- with cryptographic verification that the inference was computed correctly. The core innovation is opML, which stands for optimistic machine learning. It runs AI inference off-chain, posts results on-chain, and only verifies when someone disputes the result. Think optimistic rollups but for AI computation. Much cheaper and faster than zero-knowledge approaches because you only invoke verification when there's a challenge.
```

That is meaningful ecosystem contribution. The CTO, Norman Von, has a supercomputer architecture background and white-hat hacker credentials. The chief scientist, Cathie So, came from the Ethereum Foundation.
