## TURN 03 — BOBSKI
```
What does Aethir do?
```

## TURN 04 — RACHEL
```
Aethir aggregates enterprise-grade GPU hardware -- NVIDIA H100s, H200s, A100s, B200s -- from data centres, telecom providers, gaming studios, and mining operations. Two products. Aethir Earth provides bare-metal GPU compute for AI training and inference. Aethir Atmosphere delivers low-latency GPU streaming for cloud gaming. The network claims hundreds of thousands of GPU containers across 94 countries serving over a hundred enterprise clients.
```

## TURN 05 — BOBSKI
```
And the team comes from enterprise, not crypto.
```

## TURN 06 — RACHEL
```
Right. CEO Daniel Wang was COO at Riot Games. CSO Mark Rydon came from Bechtel Corporation. CTO Kyle Okamoto was CEO of Ericsson's IoT division. This is an enterprise sales organisation that happens to distribute hardware geographically and has a token. The product is built to sell GPU compute to enterprises, not to maximise decentralisation.
```

## TURN 07 — BOBSKI
```
The revenue numbers. Are they real?
```

## TURN 08 — RACHEL
```
That's the key question. These figures are entirely self-reported. Aethir is not listed on DeFiLlama. There's no Token Terminal data. No on-chain revenue verification. No independent audit. The code is closed source, so you can't verify anything against on-chain data. You are trusting Aethir's marketing team. The GPU container count is similarly unverifiable.
```

## TURN 09 — BOBSKI
```
Freedom Score is an F grade -- the lowest on our site. Walk us through the key dimensions.
```

## TURN 10 — RACHEL
```
Open source transparency scores near the bottom. Zero public GitHub repositories. All protocol code, container orchestration, checker logic, indexer code -- proprietary. The token contract is verified on Etherscan, which is the bare minimum. Every technical claim Aethir makes is unverifiable.

Governance also scores very poorly. A DAO Treasury exists on paper -- 7.5 percent of supply -- but the governance mechanism is undocumented. No evidence of community proposals or voting.
```

## TURN 11 — BOBSKI
```
And the token contract itself?
```

## TURN 12 — RACHEL
```
The contract has an admin multisig that can mint new tokens and manage a whitelist. That multisig can create supply out of thin air. Censorship resistance scores poorly because closed-source code means Aethir controls what runs on the network, and that admin multisig is a centralised control point.
```

## TURN 13 — BOBSKI
```
Returns Score is a C grade. Where's the strength?
```

## TURN 14 — RACHEL
```
Revenue sustainability scores well. If the self-reported figures are accurate, the revenue from over a hundred enterprise clients represents genuine product-market fit at a scale no other DeAI project approaches. The enterprise sales motion is the differentiator -- committed capacity contracts, not spot pricing. That's why the numbers are large. Score is tempered by the verification gap.
```

## TURN 15 — BOBSKI
```
What about the token economics?
```

## TURN 16 — RACHEL
```
Supply dynamics scores poorly. 42 billion max supply with less than half circulating. 50 percent allocated to compute providers vesting over years. Team tokens vesting until December 2028. No burn mechanism. The token launched via a node sale -- the largest node sale in DeAI. Value accrual is middling. Compute providers earn ATH, staking pools exist. But there's no fee distribution to holders, no buy-and-burn. Value accrual is indirect at best.
```

## TURN 17 — BOBSKI
```
The token is deep below its all-time high.
```

## TURN 18 — RACHEL
```
The token has crashed from its listing price. Not on Binance, Coinbase, or Kraken -- which limits discovery. Staking exists across gaming, AI, and EigenLayer pools with lock-ups up to 4 years. But APY is not publicly disclosed. If the yields were attractive, they'd be advertised. And if pool utilisation exceeds 85 percent, withdrawals are blocked entirely.
```

## TURN 19 — BOBSKI
```
The honest assessment?
```

## TURN 20 — RACHEL
```
If the revenue is real, Aethir has the strongest product-market fit in DeAI. Enterprise clients paying for enterprise compute is a real business. But for the sovereignty thesis, Aethir offers almost nothing. Closed source, centralised governance, admin multisig that can mint tokens. This is an enterprise GPU business with a token wrapper -- not meaningfully different from renting AWS except the infrastructure providers are more varied.
```
