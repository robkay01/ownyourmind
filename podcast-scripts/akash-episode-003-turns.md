## TURN 03 — ROB

```
So Rachel, what exactly is Akash? The elevator pitch.
```

## TURN 04 — RACHEL
```
Akash is a decentralised cloud computing marketplace built on a Cosmos SDK appchain. Providers list spare GPU, CPU, memory and storage capacity. Tenants describe workloads in a YAML-based configuration language, submit deployment orders on-chain, and providers bid via reverse auction. Lowest qualifying bid wins. Workloads run in Kubernetes containers.
```

## TURN 05 — ROB
```
So it's Airbnb for compute — people with spare capacity list it, people who need compute rent it.
```

## TURN 06 — RACHEL
```
Exactly. The chain handles order matching, escrow payments, and provider attestation. The actual compute happens off-chain. GPU marketplace launched in September 2023, and they've since added AkashML for managed inference.
```

## TURN 07 — ROB
```
What's the value proposition? Why use Akash over AWS?
```

## TURN 08 — RACHEL
```
Cost. Akash runs 50 to 85 percent cheaper than AWS, GCP, and Azure. The reverse auction model drives prices down because providers compete for your deployment. But the real differentiator is permissionless supply. Anyone with qualifying hardware can become a provider. No approval process, no geographic restrictions.
```

## TURN 10 — RACHEL

The code is complete, activation target is early to mid 2026. But at 3.15 million annual revenue against an 88 million market cap, BME can't drive meaningful deflation yet.
