## TURN 03 — BOBSKI
```
What does Golem actually do in 2026?
```

## TURN 04 — RACHEL
```
It's a peer-to-peer marketplace for computing power. Providers share spare CPU and GPU resources. Requestors submit jobs which get split into subtasks and distributed across provider nodes. The core software is Yagna -- an open-source Rust framework. Supported workloads include 3D rendering, AI inference, scientific simulations, and zero-knowledge proof generation. Pricing runs about a tenth of a cent per core-hour for CPU and one cent for GPU -- claimed 70 to 90 percent below AWS.
```

## TURN 05 — BOBSKI
```
And the team?
```

## TURN 06 — RACHEL
```
Founded by Julian Zawistowski and three co-founders. The entity is Golem Factory GmbH in Zug, Switzerland. But in 2019, Zawistowski departed to establish the Golem Foundation -- a separate non-profit. Over 40 million dollars in ETH and the original GNT token was transferred from Factory to Foundation. So now you've got two entities with shared token but divergent strategies and separate treasuries.
```

## TURN 07 — BOBSKI
```
The most interesting recent development is the Salad.com partnership, right?
```

## TURN 08 — RACHEL
```
Salad is a 200 million-plus revenue GPU cloud platform. The engineering test mirrors Salad's commercial workloads onto Golem's permissionless execution layer. If this progresses beyond testing, it's Golem's first real commercial integration at scale. But right now it's an engineering test, not a revenue contract.
```

```
Token distribution fairness is one of the highest we have scored. 82 percent to ICO participants, only 18 percent to team and company. All tokens immediately liquid. No remaining unlocks. Permissionless P2P network, anyone can join as provider or requestor without permission or KYC.
```

```
Where does it fall short on freedom?
```

```
Governance is middling. GAPs -- Golem Amendment Proposals -- exist since 2020, but in practice Golem Factory retains control over development and roadmap. No formal DAO. Data sovereignty also lands in the middle because there's no confidential computing deployed -- no TEE, no encrypted task delegation. Requestors send raw data to providers.
```

Providers earn GLM and sell immediately -- high token velocity with no sinks.
