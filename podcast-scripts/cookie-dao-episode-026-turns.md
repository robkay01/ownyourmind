## TURN 03 — BOBSKI
```
What does cookie.fun actually do?
```

## TURN 04 — RACHEL
```
It is a data analytics platform that tracks over 1,500 AI agents across crypto. It ingests 7 terabytes of live on-chain and social data to calculate proprietary metrics -- most notably Mindshare, which measures an AI agent's share of social conversation volume. That metric has become a widely referenced standard in the DeFAI space.

They also offer Smart Following for tracking influencers and smart money, Cookie Data APIs for AI agents and dapps, Cookie Snaps for gamified social engagement rewards, and a trading interface launched in May 2025.
```

## TURN 05 — BOBSKI
```
So it is basically the DeFiLlama of AI agents.
```

## TURN 06 — RACHEL
```
That is the pitch, and it is a fair comparison. DeFiLlama, Nansen, DexScreener -- these are centralised and valuable. Cookie.fun has real product-market fit: 60 thousand users, 20 thousand within the first 48 hours of launch, 5 million COOKIE tokens locked for premium access. The Binance listing in January 2025 pushed the market cap to a substantial level.
```

## TURN 07 — BOBSKI
```
But the token has collapsed from its all-time high -- a steep decline that most holders have felt.
```

## TURN 08 — RACHEL
```
Yes. And the vesting schedule is a big part of why. The vesting was unusually aggressive -- 1-month cliff plus 6-month linear, compared to the industry standard of 12-month cliff plus 24 to 36 months. Most insider tokens unlocked by January 2025 -- the exact month of the all-time high. That is not a coincidence.
```

## TURN 09 — BOBSKI
```
One of our lowest Freedom Scores ever. Walk me through it.
```

## TURN 10 — RACHEL
```
Infrastructure decentralisation scores very poorly. The platform, data pipeline, indexing engine, API, and Mindshare algorithm all run on Cookie3's servers. If they shut down, cookie.fun stops. No independent nodes, no permissionless data providers.

Governance decentralisation also scores very poorly. Token voting is claimed but no public Snapshot space, no governance forum, no on-chain voting history could be found. Cookie3 and core contributors make all operational decisions.

Data sovereignty scores poorly. The Mindshare metric and agent scores are proprietary with no verifiable methodology on-chain. The entire data layer is controlled by Cookie3.
```

## TURN 11 — BOBSKI
```
And zero open source code.
```

## TURN 12 — RACHEL
```
Zero. The cookie3 GitHub organisation has zero public repositories. The entire codebase -- data pipeline, indexing engine, API, Mindshare algorithm -- is proprietary. You cannot audit, fork, or independently verify any technology claims. Open source transparency scores very poorly.
```

## TURN 13 — BOBSKI
```
The Returns Score is weak. What is working on the economics side?
```

## TURN 14 — RACHEL
```
Token utility is decent. Premium analytics require locking COOKIE. API access requires the token. Multi-Airdrop Farming earns a share of partner reward pools. Snaps participation. If you want the data, you need the token -- that creates genuine demand beyond speculation.

Value accrual is reasonable. Two burn mechanisms -- 50 percent of API fees burned, plus 10 percent of MAF pool locks burned. The dual burn is structurally sound.
```

## TURN 15 — BOBSKI
```
But?
```

## TURN 16 — RACHEL
```
Actual API revenue is not disclosed, making it impossible to verify the real burn impact. Revenue sustainability scores poorly. No annual revenue figure, no DeFiLlama data, no Token Terminal data. The product has real usage but revenue sustainability is a black box.
```

## TURN 17 — BOBSKI
```
Key risks?
```

## TURN 18 — RACHEL
```
Total centralisation is the headline risk. X slash Twitter API dependency -- if Twitter changes API terms, the social data feed breaks. Competition from Kaito AI and DexScreener expanding into AI agent analytics. And narrative dependency -- if the AI agent narrative fades, so does the analytics demand.
```

## TURN 19 — BOBSKI
```
The honest take?
```

## TURN 20 — RACHEL
```
Cookie.fun is a good product. Real users, first-mover advantage, the Mindshare metric has genuine adoption. But it is a centralised data company with a token, branded as a DAO. Calling it a DAO is misleading. Your data access, your analytics, your API feeds all depend on one company in Tallinn, Estonia continuing to operate. For a sovereignty-focused listener, this scores an F on freedom for good reason.
```
