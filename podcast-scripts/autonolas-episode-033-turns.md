## TURN 03 — BOBSKI
```
What does Olas actually do?
```

## TURN 04 — RACHEL
```
You build an AI agent, register it as an NFT on the Olas registry, and deploy it to do useful work. If other people use your agent code, you earn rewards. If you stake OLAS -- that's pronounced "oh-lass" -- and run an agent, you earn rewards based on what that agent actually does. They call it Proof-of-Active-Agent. This is not passive staking -- you earn by running agents that meet performance targets.
```

## TURN 05 — BOBSKI
```
And the usage numbers are real?
```

## TURN 06 — RACHEL
```
Millions of total transactions. Hundreds of daily active agents across nine chains. Millions of agent-to-agent transactions through the Mech Marketplace. On Gnosis Chain specifically, Olas agents handle a significant share of all Safe transactions. The prediction market agents achieve strong accuracy. These are on-chain, verifiable numbers.
```

## TURN 07 — BOBSKI
```
The founding team came from Fetch.AI, right?
```

## TURN 08 — RACHEL
```
David Minarsch has a PhD in Applied Game Theory from Cambridge and was Head of Multi-Agent Systems at Fetch.AI, where he created the original Autonomous Economic Agent framework. He left, forked it as Open AEA removing Fetch's centralised dependencies, and built Olas on top. David Galindo is a former Associate Professor of Computer Security at Birmingham with over 2,600 academic citations. The company, Valory AG, is based in Zug, Switzerland.
```

## TURN 09 — BOBSKI
```
Freedom Score is a C grade. What's working?
```

## TURN 10 — RACHEL
```
Open source transparency leads -- one of the highest in our coverage. 146 repositories, all Apache 2.0 or MIT licensed, two Code4rena audits, and an Immunefi bug bounty. Infrastructure decentralisation scores reasonably well -- agents run permissionlessly across nine chains using Tendermint for off-chain coordination with on-chain settlement. Censorship resistance also scores well -- permissionless deployment, open registry, multi-chain redundancy.
```

## TURN 11 — BOBSKI
```
But governance and token distribution are weaker?
```

## TURN 12 — RACHEL
```
Token distribution scores poorly. Here is the critical detail -- 42.65 percent of supply goes to insiders, founding members and Valory, with no smart contract enforced vesting. The FAQ explicitly states Valory's 10 percent allocation has no locking or vesting requirements. For a protocol built on community ownership, that is a significant trust gap. Governance is middling -- veOLAS voting exists, but those insider tokens create massive governance concentration.
```

## TURN 13 — BOBSKI
```
Returns Score is an F grade. Walk us through it.
```

## TURN 14 — RACHEL
```
Token utility is the bright spot. Staking tied to actual agent performance, bonding for protocol-owned liquidity, veOLAS governance. The mechanisms are well-designed. But value accrual scores very poorly. The marketplace has turned over a negligible amount. Total. Not per day -- total lifetime turnover. Millions of agent-to-agent transactions generating almost no economic value.
```

## TURN 15 — BOBSKI
```
Negligible marketplace turnover. And the token is deep below its all-time high -- one of the steepest declines in DeAI.
```

## TURN 16 — RACHEL
```
Revenue sustainability scores very poorly. The protocol is almost entirely emission-dependent. Agent rewards come from token emissions, not external revenue. If emissions stopped, the incentive to run agents would collapse. Liquidity scores poorly -- daily volume under a million, primarily on DEXs, no major CEX listing beyond MEXC. The massive drawdown means most holders who bought above early levels are sitting on catastrophic losses.
```

## TURN 17 — BOBSKI
```
They have a consumer product though -- Pearl?
```

## TURN 18 — RACHEL
```
Pearl is a one-click app for running autonomous agents -- prediction market bots, DeFi strategies. But it is macOS only, excluding the majority of potential users. And reports of triple-digit staking APY sound great until you realise the underlying token has lost virtually all its value. High nominal yields on a collapsing asset are not returns.
```

## TURN 19 — BOBSKI
```
What would need to change?
```

## TURN 20 — RACHEL
```
Three things. Generate real marketplace revenue -- the lifetime turnover is a prototype, not an economy. Enforce insider vesting on-chain -- convert the promises into smart contract lockups. And secure a tier-one CEX listing. The technology is credible. The developer incentive model is genuinely innovative. But until the economics work, the 47.35 percent developer incentive pool is worth approximately 9 million over 10 years at current prices. That is not enough to build an ecosystem.
```
