## TURN 03 — BOBSKI

```
What makes IoTeX architecturally different from other L1s?
```

## TURN 04 — RACHEL
```
Five core components. The base chain uses Roll-DPoS consensus -- DPoS combined with PBFT and verifiable random functions. 36 community-elected consensus delegates, 24 randomly selected each epoch. 2.5-second block time, 1-block finality, up to 1,000 transactions per second.

But the real differentiators are ioID and W3bstream. ioID is a decentralised identity framework for machines and AI agents using W3C standards, represented as NFTs on-chain. W3bstream is an off-chain computation layer that uses ZK proofs to verify device data without exposing raw data.
```

## TURN 05 — BOBSKI
```
So a physical device gets an on-chain identity as an NFT?
```

## TURN 06 — RACHEL
```
Exactly. When you register a device, an ioID NFT is minted. You provably own the device on-chain and control its data and rewards. No other chain offers this. It is device-level data sovereignty implemented at the protocol level, not as an application.
```

## TURN 08 — RACHEL

```
Data sovereignty is the strongest dimension. ioID gives devices self-sovereign identity via W3C DIDs and verifiable credentials as NFTs. W3bstream ZK proofs verify device data without exposing raw data. Device keys stored on-device. Among the strongest data sovereignty architectures in DePIN.

Infrastructure decentralisation scores well. Own L1 with 36 consensus delegates, 60-plus total, permissionless node operation. Mainnet has operated continuously since April 2019. The founder, Raullen Chai, was personally committing code through March 2026.
```
