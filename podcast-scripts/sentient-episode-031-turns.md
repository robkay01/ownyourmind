## TURN 03 — BOBSKI

What is Sentient building?
```

## TURN 04 — RACHEL
```
Three things. First, OML -- Open, Monetisable, Loyal -- a cryptographic standard that embeds undetectable fingerprints into language models. If someone copies your model, you can prove ownership. If someone uses it, you earn per-invocation fees via smart contracts. This is a peer-reviewed paper, not a marketing claim.

Second, ROMA -- a recursive multi-agent framework with nearly 5,000 GitHub stars. Third, The GRID -- an open-source AGI network that launched in testnet August 2025 with 40 plus agents and 50 plus data sources. Mainnet is planned for Q2 2026.
```

## TURN 05 — BOBSKI
```
The founding team is unusual for crypto.
```

## TURN 06 — RACHEL
```
Pramod Viswanath is the Forrest G. Hamrick Professor of Engineering at Princeton and co-invented Flash OFDM -- the technology underlying 4G mobile networks. Himanshu Tyagi is a professor at the Indian Institute of Science. Sandeep Nailwal co-founded Polygon. Kenzi Wang co-founded Symbolic Capital. You have world-class academics paired with crypto-native operators. That combination is rare.

But governance scores poorly. It's planned but not active. All current decisions are made by the founding team. And infrastructure also scores poorly because GRID mainnet is not live yet.

Returns Score is also a D grade. What's the picture?

For the sovereignty thesis, does OML matter?

The Dobby model family on HuggingFace includes explicitly uncensored fine-tunes. That is ideologically aligned with sovereignty. But until GRID mainnet proves the system works end-to-end, it is architecture and aspiration.

Everything hinges on Q2 2026.
